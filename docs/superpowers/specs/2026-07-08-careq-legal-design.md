# careq-legal 설계 문서

작성일: 2026-07-08

## 목적

CareQ 서비스의 법적 문서(이용약관, 개인정보처리방침)를 호스팅하는 정적 웹페이지.
앱 심사 및 회원가입 화면에서 링크로 연결되는 용도이며, 주로 모바일 웹뷰에서 열린다.

## 범위

- 이용약관 페이지 1개
- 개인정보처리방침 페이지 1개
- 두 문서로 연결되는 랜딩 페이지 1개
- 문서 본문 텍스트는 사용자가 별도 제공 (제공 전까지는 자리 표시자)

## 기술 결정

- **순수 정적 HTML + CSS, 빌드 도구 없음.** 문서 2개짜리 프로젝트에 프레임워크나
  정적 사이트 생성기는 과투자. 수정 시 HTML 편집 후 push만 하면 된다.
- **JavaScript 없음.**
- **배포: GitHub Pages**, `main` 브랜치 루트에서 서빙.

## 파일 구조

```
careq-legal/
├── index.html      # 문서 목록 랜딩
├── terms.html      # 이용약관
├── privacy.html    # 개인정보처리방침
├── style.css       # 공용 스타일
└── README.md       # 프로젝트 설명 + 배포 URL
```

## 페이지 디자인

- 읽기 중심: 본문 최대 너비 720px 중앙 정렬, 시스템 폰트 스택, 행간 1.7
- 라이트/다크 모드 대응 (`prefers-color-scheme`)
- 각 문서 페이지: 상단에 문서 제목과 시행일, 하단에 다른 문서로 가는 링크
- 언어: 한국어 (`lang="ko"`)

## 배포 절차

1. GitHub에 저장소 생성 후 push
2. Settings → Pages → Source: `main` 브랜치, `/ (root)` 지정
3. `https://<계정>.github.io/careq-legal/` 로 접근

## 에러 처리 / 테스트

정적 문서 페이지로 런타임 로직이 없다. 확인 항목은 두 가지:
브라우저(모바일 폭 포함)에서 렌더링 확인, 페이지 간 링크 동작 확인.
